# Online Quiz App (Java Mini Project)
# Objective
  To create a simple console-based quiz application in Java that:
  Displays multiple-choice questions
  Accepts user input
  Calculates and displays the final score
# Features
  Supports multiple quiz questions
  Each question has 4 options (A, B, C, D)
  User enters answers via keyboard
  Automatic score calculation
  Single Java file implementation

# Main Program Steps
  Create a list of questions using ArrayList
  Display each question and options
  Accept user input (A/B/C/D)
  Check if the answer is correct
  Calculate and display the final score
